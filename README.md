# BookXNote
BookXNote 是一款优秀的读书笔记软件，可以在读书的同时快速做读书笔记，支持PDF、epub、mobi、txt、docx等格式的电子书。

# 关于本插件
本插件是BookXNote的插件，可以把保存在BookXNote中的笔记一键同步到Obsidian中
